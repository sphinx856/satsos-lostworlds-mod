Sphinx Savegame Editor

Sphinx Savegame Editor (SSE) is an MDI type tool that allows you to modify the savegames from Sphinx game, in this page you can download the latest version of this editor. This program allow you to:

  • Modify objectives.
  • Modify items (min, max values, and flags).
  • Export the values from one savegame and apply them to other one.

Apart from this, you can create your own scripts of values and apply it to your savegames. Take in mind that those scripts, only change the values of the objectives and items that are stored in the savegame. Is not possible to add or delete bytes from the savegame. Another important feature is that before applying a script, the editor always ask you to create a backup file of the original one. This way, if something goes wrong, you will not lose your original savegame.

Important note: When you start the program for the first time, you need to set the hashcodes.h file location in the program settings. To do that you have to go to: View->Preferences->Hashcodes here you have two options:

  • Use a static location, for example X:\Sphinx\Albert\hashcodes.h if you have the Authoring Tools DLC installed.
  • Use copy from parent folder, that means that you can have the hashcodes.h file in the same location that the .exe is.
